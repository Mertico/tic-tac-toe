# Реализовать игру **Крестики Нолики**

```js
X|O|
-+-+-
 |X|O
-+-+-
O| |X
```

## Требования к реализации

**1. Пользовательский интерфейс**

- [ ] Интерактивная доска 3x3 с поддержкой поочередного ввода X/O
- [ ] Визуализация текущего состояния игры (подсветка последнего хода)
- [ ] Экран результатов с объявлением победителя (X/O) или ничьей
- [ ] Кнопка перезапуска игры

**2. Логика игры**

- [ ] Алгоритм проверки победной комбинации после каждого хода
- [ ] Обработка ничьей (все клетки заполнены без победителя)
- [ ] Поддержка перезапуска игры без перезагрузки страницы

**3. Онлайн-режим (WebSocket)**

- [ ] Серверная часть:
  - Генерация уникального ID комнаты и ссылки на неё
  - Синхронизация состояния между участниками
  - Проверка допустимости ходов
  - Обработка дисконнектов
- [ ] Клиентская часть:
  - Создание/присоединение к комнате по ссылке
  - Отображение статуса соперника (ожидание/ход/победа)
  - Блокировка интерфейса при ходе соперника
  - Автоматический рестарт (по согласованию или таймеру)

**4. Дополнительно**

- [ ] Система быстрого старта (автоматический подбор соперника)
- [ ] Валидация входных данных игроков
- [ ] Логирование игровых событий
- [ ] Таймер хода
